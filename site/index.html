直拼ZhiPin

所输即所得的拼音输入法 — 不模糊、不猜测、尊重你敲下的每一个字母。

下载 macOS 版ZhiPin-macos.pkg · Apple Silicon & Intel 下载 Windows 版ZhiPin-windows-setup.exe · 64 位

与搜狗输入法有什么不同？

  • 不猜测：搜狗会纠错、模糊音、联想；直拼只精确匹配你敲下的拼音， 拼错就不出字，绝不替你做决定
  • 不联网：搜狗依赖云端候选与账号同步，你的输入会上传； 直拼完全离线，没有一行联网代码
  • 词库透明：自造词是一个可读的文本文件，每条都能在候选栏点 ✕ 永久删除； 不是一个不可见的个人模型
  • 确定性：同样的按键永远得到同样的候选序列（除了你自己教它的词）
  • 取舍：搜狗的长句转换准确率和新词热词更强； 直拼胜在可控、可审计、尊重输入

特点

  • 精确匹配：无模糊音（zh ≠ z），拼错就不出字
  • 整句输入：一次最多 10 个汉字，基于 46 万条词库转换
  • 末字补全：链式输入中只补全正在敲的那一个字，绝不多猜
  • 自造词组：你选过的组合保存在本机，下次同样拼音直接置顶（标 ★）； ⌃⌫ / Ctrl+Del 可永久删除
  • Shift 单击切换中 / 英文
  • 完全离线，词库与自造词只存在你的电脑上

macOS 安装

  • 双击 pkg 安装（未签名：如被拦截，在「系统设置 → 隐私与安全性」中点“仍要打开”）
  • 系统设置 → 键盘 → 输入法 → 编辑 → + → 简体中文 → 直拼
  • 如列表中未出现，注销并重新登录一次

Windows 安装

  • 运行安装程序（SmartScreen 提示时选择“仍要运行”）
  • 确保已添加「中文(简体，中国)」语言，输入法列表中选择 直拼 ZhiPin
  • Win+空格 切换到直拼

按键

  • 1–9 选字 · 空格 选中当前高亮 · 回车 上屏原始字母
  • -/= 或 PgUp/PgDn 翻页 · ↑/↓ 移动高亮
  • ' 手动切分音节（xi'an → 西安）
